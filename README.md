# Localizer Landing Page

مشروع Landing Page منفصل لعرض الكاردات والتشريعات.

## المميزات

- عرض الكاردات من API
- تصميم متجاوب
- واجهة مستخدم حديثة

## التثبيت

```bash
npm install
```

## إعداد البيئة

أنشئ ملف `.env`:

```env
VITE_API_URL=http://localhost:8000/api
```

## التشغيل

```bash
npm run dev
```

الخادم سيعمل على: `http://localhost:5173`

## البناء

```bash
npm run build
```
